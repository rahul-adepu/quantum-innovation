# Quantum Innovation - Cybersecurity Website

A professional, enterprise-grade frontend website for Quantum Innovation, a cybersecurity and AI solutions company.

## Tech Stack

- **React** - UI library
- **Vite** - Build tool and dev server
- **Tailwind CSS** - Utility-first CSS framework
- **Framer Motion** - Animation library
- **Lucide React** - Icon library

## How to Clone and Run

```bash
# Clone the repository
git clone https://github.com/rahul-adepu/quantum-innovation.git

# Navigate to the project directory
cd quantum-innovation

# Install dependencies
npm install

# Run the development server
npm run dev
```

## Features

- Fully responsive design (Mobile, Tablet, Desktop)
- Clean, professional UI/UX
- Enterprise-grade cybersecurity theme
- Smooth navigation
- Accessible components
